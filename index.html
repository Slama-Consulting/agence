Comment ça marche

  1. 01

    Choisir un agent

    Parcourez la galerie et cliquez sur « Recruter » pour ouvrir le dossier de l'agent.

  2. 02

    Installer

    Une seule commande shell. La distribution est servie depuis GitHub Pages, le code source reste privé.

    Configurer

    Ajoutez l'agent à votre client MCP : Claude Code, Copilot ou Cursor. Et l'enquête peut commencer.
